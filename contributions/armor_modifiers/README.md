# Armor Modifiers

Material modifiers for armor per [docs/armor_system.md](../../docs/armor_system.md). Applied to a base armor template to produce final stats.

**Fields:** `modifier_id` or `id`, `name`, `dr_bonus`, `weight_modifier`, `hp_bonus`, optional `notes`.

Example: Bone modifier with `dr_bonus: 0`, `weight_modifier: -1`, `hp_bonus: -15` makes lighter, brittler armor.
